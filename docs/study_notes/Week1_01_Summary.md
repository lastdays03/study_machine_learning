# 1주차 Session 1: 핵심 개념 요약 (Reflection)

> **Q1. 과적합(Overfitting)이란 무엇인가요?**

**A.**
모델이 **훈련 데이터(Training Data)**의 패턴을 지나치게 상세하게 학습하여, 학습하지 않은 **테스트 데이터(Test Data)**나 새로운 데이터에 대해서는 오히려 예측 성능이 떨어지는 현상을 말합니다.
즉, 모델이 일반화(Generalization)되지 못하고 '암기'해버린 상태입니다. (비유: 문제집 답만 달달 외워서 조금만 바뀐 시험 문제는 못 푸는 것)

---

> **Q2. Feature와 Label의 차이는 무엇인가요?**

**A.**
*   **Feature (특성/입력변수, $x$)**: 모델이 예측을 수행하기 위해 사용하는 **단서(Input)**입니다.
    *   예: 아파트 가격 예측 시 '평수', '방 개수', '지하철 거리' 등.
*   **Label (정답/타겟변수, $y$)**: 모델이 맞혀야 하는 **정답(Output)**입니다.
    *   예: 아파트 가격 예측 시 '실거래가'.
